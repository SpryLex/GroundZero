# Ekayatraa — On-Demand Consultancy Marketplace

Ekayatraa connects clients with domain experts through a gig-based, fully remote consultancy platform. Users can browse expert profiles, book sessions, and get on-demand advice across multiple domains.

## Tech Stack

- **Frontend:** React, Vite
- **Backend:** Node.js, Express

## Project Structure
GroundZero/

├── frontend/    # React app

├── backend/     # Express server

## Getting Started

### Prerequisites
- Node.js installed

### Run the backend
```bash
cd backend
npm install
node index.js
```

### Run the frontend
```bash
cd frontend
npm install
npm run dev
```
